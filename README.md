# Nest.js modules

### Cassandra module
```bash
npm install --save @mich4l/nestjs-cassandra cassandra-driver
```

### Redis module
```bash
# pnpm:
pnpm add @mich4l/nestjs-redis ioredis
# npm:
npm install --save @mich4l/nestjs-redis ioredis
# yarn:
yarn add @mich4l/nestjs-redis ioredis
```
