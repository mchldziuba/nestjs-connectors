## ClickHouse module for Nest.js
ClickHouse module based on official client `@clickhouse/client`.

### Installation
```shell
# pnpm:
pnpm add @md03/nestjs-clickhouse @clickhouse/client
# npm:
npm install @md03/nestjs-clickhouse @clickhouse/client
```
